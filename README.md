# Seven Layers of OSI Models

There Are 7 types of Networking Layers :

7. APPLICATION LAYER: = The application layer is the top-most layer (Layer 7) in the Open Systems Interconnection (OSI) model & the Internet Protocol Suite (TCP/IP) that serves as the direct interface between user software applications (like web browsers or email clients) and the underlying network.
6. PRESENTATION LAYER: = The presentation layer is the 6th layer of the OSI model, acting as a translator between the application layer & lower network layers. It ensures that data is in a usable, standardized format by handling syntax, encryption, & compression, making it compatible for different systems.
5. SESSION LAYER: = The Session Layer is the 5th layer of the OSI model, responsible for establishing, managing, synchronizing, & terminating connections (sessions) between applications on different devices. It acts as a dialogue controller, ensuring data exchange is organized, bidirectional or unidirectional, and resumes correctly after interruptions.
4. TRANSPORT LAYER: = The Transport Layer (Layer 4) is a core component of the OSI model & TCP/IP suite that handles end-to-end communication, ensuring reliable and organized data transfer between applications on different networked hosts. It breaks data into segments, manages flow control, detects errors, and uses port numbers for process-to-process delivery.
3. NETWORK LAYER: = The network layer is the third layer of the OSI model, responsible for facilitating data transfer between different networks by managing logical addressing (IP addresses) and routing data packets along the best path. It moves data from a source host to a destination host, even across intermediate networks.
2. DATA LINK LAYER: = The Data Link Layer (Layer 2) is the second layer of the OSI model, responsible for node-to-node data transfer between directly connected devices within the same network. It ensures reliable, error-free communication by formatting data into frames, handling physical addressing (MAC addresses), and managing access to the physical medium.
1. PHYSICAL LAYER: = The Physical Layer (Layer 1) is the lowest level in the OSI model, responsible for the physical, mechanical, & electrical transmission of raw data bits (\(0\)s & \(1\)s) across a network medium, such as cables or radio waves. It converts digital signals into electrical, optical, or radio signals & defines hardware specifications like pin layouts, voltages, & cable types.
