Application Layer = The application layer is the top-most layer (Layer 7) in the Open Systems Interconnection (OSI) model and the Internet Protocol Suite (TCP/IP) that serves as the direct interface between user software applications (like web browsers or email clients) and the underlying network

5. Session Layer
